# bulk-RNA-seq-tutorial-codes
bulk RNA-seq tutorial codes. Replicating (parts of) Yin et al. (2023). 

Note: You will need the top 19 samples (SRR24860152 to 24860170) from https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE239603 to replicate the figures. [vignette.ipynb](https://github.com/dowchbag/bulk-RNA-seq-tutorial-codes/blob/main/vignette.ipynb) only contains the top 4 samples, so edit your code accordingly. 

Upon completion, you should be able to replicate the figures specified by the name of the .R file. Run [figure_1f.R](https://github.com/dowchbag/bulk-RNA-seq-tutorial-codes/blob/main/figure_1f.R) first because it calls the necessary pacakges and defines the data for the other two scripts.
